# Começando com NodeJS em 2022 - Rocketseat

## Desenvolvendo um CRUD simples com express

### Funções:
    - Criar registro
    - Consultar registro
    - Alterar registro
    - Excluir registro
    - Gerar arquivo contendo os registros

### Libs utilizadas:

    - nodemon@2.0.16
    - lib/fs.js

### Versão do NodeJS:
    - v16.13.2
